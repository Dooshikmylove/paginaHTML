FORMULARIOS

Los formularios en HTML son una forma de recopilar datos del usuario y enviarlos a un servidor para su
procesamiento. Un formulario típico consta de varios elementos, como:
1. *Etiqueta `
`*: Define el formulario y especifica el método de envío (GET o POST) y la acción (URL del servidor que procesará los datos).
2. *Campos de entrada*:
- *`input`*: Permite al usuario ingresar texto, números, fechas, etc.
- *`textarea`*: Permite al usuario ingresar texto en un área más grande.
- *`select`*: Presenta una lista desplegable de opciones.
- *`checkbox`*: Permite al usuario seleccionar una o varias opciones.
- *`radio`*: Permite al usuario seleccionar una opción entre varias.
3. *Etiquetas*:
- *`label`*: Asocia un texto con un campo de entrada.
4. *Botones*:
- *`input type="submit"`*: Envía el formulario.
- *`input type="reset"`*: Restablece los campos del formulario.
5. *Atributos*:
- *`name`*: Especifica el nombre del campo.
- *`type`*: Especifica el tipo de campo (texto, contraseña, etc.).
- *`value`*: Especifica el valor predeterminado del campo.
