SEMANTICAS

Las semánticas de HTML se refieren al uso de elementos HTML para describir el significado y la estructura del
contenido de una página web, en lugar de solo su presentación visual.
La semántica HTML implica utilizar elementos HTML para identificar el propósito y la función de cada parte del contenido, como:
- < header> para la cabecera de la página o sección
- < nav> para la navegación
- < main> para el contenido principal
- < section> para secciones de contenido
- < article> para artículos o entradas de blog
- < aside> para contenido relacionado pero secundario
- < footer> para el pie de página
Esto permite que los navegadores, los motores de búsqueda y las herramientas de accesibilidad entiendan
mejor la estructura y el significado del contenido, lo que puede mejorar:
- La accesibilidad para usuarios con discapacidades
- La indexación y el ranking en motores de búsqueda
- La legibilidad y la comprensión del contenido
- La capacidad de respuesta y la adaptabilidad en diferentes dispositivos
Además, las semánticas HTML también incluyen el uso de atributos como `alt` para describir imágenes, `lang`
para especificar el idioma del contenido, y `aria-*` para mejorar la accesibilidad.
El uso de semánticas HTML es una buena práctica para desarrollar páginas web modernas y accesibles.
