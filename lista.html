LISTA

En HTML, podemos crear una lista ordenada usando la etiqueta < ol> .
La etiqueta ol representa una lista ordenada. Dentro de cada uno de los elementos de la lista ordenada < ol> y
< /ol> , tenemos que definir los elementos de la lista. Podemos definir los elementos de la lista usando la etiqueta < li> .
Estructura HTML para una lista ordenada:
< ol>
< li> < /li>
< li> < /li>
< li> < /li>
</ol>
