AUDIO

La etiqueta HTML "audio" tiene varios atributos que podemos utilizar:
Atributo Descripción
src Audio a reproducir. Obligatoria si no contiene etiquetas "source".
Un primer ejemplo muy básico para colocar un audio en nuestra página web sería utilizar el atributo src para
indicar nuestro archivo multimedia de audio:HTML
audio src="audio.mp3"audio>
Respecto al formato de audio, hemos indicado un archivo .mp3, que tiene muy buen soporte, pero existen otros
formatos que también podrían ser apropiados, como .ogg o .opus.
