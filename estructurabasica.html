ESTRUCTURA BASICA

La estructura básica de HTML (Hypertext Markup Language) consta de los siguientes elementos:
1. *Doctype*: Declara el tipo de documento y la versión de HTML utilizada.
2. *html*: Elemento raíz del documento, que contiene todos los demás elementos.
3. *head*: Contiene metadatos sobre el documento, como el título, la codificación de caracteres, y enlaces a hojas de estilo o scripts.
4. *title*: Establece el título del documento, que se muestra en la barra de título del navegador.
5. *body*: Contiene el contenido del documento, como texto, imágenes, enlaces, formularios, etc.
6. *Elementos de contenido*: Son los elementos que se utilizan para agregar contenido al documento, como:
- *p*: Párrafo de texto.
- *h1-h6*: Títulos de diferentes niveles.
- *img*: Imagen.
- *a*: Enlace.
- *ul*, *ol*, *li*: Listas desordenadas, ordenadas y elementos de lista.
- *div*: Contenedor genérico para agrupar elementos.
- *span*: Contenedor inline para agrupar elementos.
7. *Atributos*: Son valores adicionales que se pueden agregar a los elementos para proporcionar más información o funcionalidad.
8. *Etiquetas de cierre*: Son las etiquetas que cierran un elemento, y se denotan con una barra inclinada (/) antes del nombre del elemento.
La estructura básica de un documento HTML sería:

!DOCTYPE html
html
head
titleTítulo del documento /title
/head
body
/body
/html
Es importante mencionar que esta es una estructura básica, y que existen muchos otros elementos y atributos
que se pueden utilizar para crear documentos HTML más complejos
