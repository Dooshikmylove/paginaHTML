HIPERVINCULOS

Los hipervínculos en HTML son enlaces que permiten a los usuarios navegar entre páginas web o dentro de
una misma página. Se crean utilizando la etiqueta <a> y se pueden utilizar para:
- Enlazar a otra página web
- Enlazar a una sección específica dentro de una página
- Enviar un correo electrónico
- Llamar a un número de teléfono
- Descargar un archivo
Atributos de la etiqueta <a>:
- < herf> : especifica la URL del destino del enlace
- < title> : especifica el texto que se muestra cuando se pasa el mouse sobre el enlace
- < target> : especifica dónde se abre el enlace (_self, _blank, _parent, _top)
- < rel> : especifica la relación entre el documento actual y el documento enlazado
Ejemplos de hipervínculos:
- Enlace a otra página web: Visitar sitio web
- Enlace para enviar un correo electrónico: Enviar correo
- Enlace para llamar a un número de teléfono: Llamar
- Enlace para descargar un archivo: Descargar archivo
Los hipervínculos se pueden estilizar utilizando CSS para cambiar su apariencia y comportamiento
